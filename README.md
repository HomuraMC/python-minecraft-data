
python-minecraft-data
=====================

Provide easy access to [minecraft-data](https://github.com/PrismarineJS/minecraft-data) in python

Installation
------------
From Repo:

``python -m pip install git+https://github.com/HomuraMC/python-minecraft-data``

Usage
-----
